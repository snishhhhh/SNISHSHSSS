# Restaurant Management System

# About the Project

This project is a Restaurant Management System developed using Python. It is a menu-driven application that allows users to manage customers, reservations, menu items, and customer bills. Customer information can also be saved and loaded using CSV files.


## Features

 View restaurant menu
 Register customers
 View customer details
 Make reservations
 Generate bills
 View order history
 Add, modify, search, and delete menu items
 Save and load customer data using CSV files

## Files

 Main.py: Runs the program
 menu.py : Restaurant menu
 customer_data.py : Customer management
 reservation.py : Reservation system
 generate_bill.py : Bill generation
 file_manager.py : Save and load customer data
 customers.csv : Stores customer information



## How to Run

1. Open the project folder.
2. Run `Main.py`.
3. Follow the menu displayed on the screen.
